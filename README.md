# 🤖 AI Interview Report Generator

An AI-powered full-stack interview preparation platform that analyzes your resume, job description, and self-description to generate personalized interview reports with actionable insights.

## 📌 Features

- 🔐 **JWT Authentication** — Secure login with protected routes and personalized report history
- 📄 **Smart Analysis** — Analyzes job description, resume, and self-description using Gemini AI
- 📊 **Match Score** — See how well your profile matches the job
- 🧠 **Skill Gap Detection** — Identifies missing skills based on the job requirements
- ❓ **Interview Questions** — Generates relevant technical and behavioral questions
- 🗺️ **Learning Roadmap** — Provides a personalized roadmap to bridge the gaps
- 📁 **Report History** — Save and revisit all your previously generated reports
